What this tool does

When you enter a domain and click "Launch Domain Links", this tool will open multiple tabs with searches across various platforms:

  • Google site search
  • GitHub domain search
  • Wayback Machine (historical snapshots)
  • Shodan (exposed services)
  • SecurityTrails (DNS history and subdomains)
  • Censys (certificates and infrastructure)
  • crt.sh (certificates)

Note: Your browser may block pop-ups the first time you use this tool. You'll need to allow pop-ups for this site for the tool to work properly. Also, shodan won't work unless you're logged in due to filter restrictions.
